
# Landlubbers (L9s) 

- __Why__  
Because the waves caused by gaps in an otherwise automated pipeline make people seasick. 

- __What__  
K8s orchestration.   
Immutable, cryptographically secured, and contractable.   
Which is to say L9s is enterprise orchestration. 

I would argue the popularity of containerization is neither in its use of virutalization nor its immutable-ization,   
nor any other of its technical gems, but rather in its combined effect of closing more gaps in the infra-as-code pipeline.    
The spawn of code that spawns infrastructure is as much a bifurcation as was code that spawns processes on one machine.  
Too few realize this, but enough so the efforts to write code that manages such code are quite active.   


But no one is minding the humans. They continue to orchestrate in this new domain as they did in the old.  
They form tribes. The defining instinct of which is to protect their territory.   
In the new domain, this manifests as gaps in the automation circuitry.   
Being purposeful, such gaps either persist or respawn elsewhere.  
The Open Source banner merely distracts from this dynamic.  
L9s rewires it, allowing the automation circuits to close.
